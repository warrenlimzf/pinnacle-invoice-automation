# Windows users

The full guide now covers both Mac and Windows in one place: see **`README.md`**.

Quick version for Windows:
1. Double-click **`setup.bat`** once.
2. Double-click **`run_watcher.bat`** and leave it open.
3. Drop statement PDFs into `banks\LGT\inbox\`, `banks\BoS\inbox\`, `banks\UBS\inbox\`.
4. Results: `output\fees_master.xlsx` and `banks\<bank>\<bank>_verification.docx`.

You do **not** need to delete old files first — old files are never re-processed.
